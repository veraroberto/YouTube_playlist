# YouTube playlist Organizer

## Things to know
1. The first thing you would have to do to use this repository is to generate a YouTube API and create the file API_KEY.py
2. The YouTube API is free to use, but it has a daily limit of 10,000 quotas. See [Quota Calculator
](https://developers.google.com/youtube/v3/determine_quota_cost).
3. This project is still in progress and I would be adding more features
